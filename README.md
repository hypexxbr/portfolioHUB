## 📋 Sobre o Projeto

O **PortfolioHUB** é uma plataforma de portfólio pessoal que integra:

- **Google Sites** como interface pública de apresentação
- **GitHub** como repositório central de versionamento e documentação
- **Google Gemini (IA)** como ferramenta de apoio na implementação

O projeto cobre o ciclo completo de implantação: planejamento, configuração, segurança, colaboração, testes e apresentação final.

## 🚀 Como Contribuir

Contribuições são bem-vindas! Leia o [CONTRIBUTING.md](./CONTRIBUTING.md) para entender o processo de colaboração.

Em resumo:
1. Faça um **fork** do repositório
2. Crie uma **branch** descritiva: `git checkout -b melhoria/nome-da-melhoria`
3. Faça suas alterações e **commit**: `git commit -m "feat: descrição da melhoria"`
4. Abra um **pull request** detalhando o que foi feito
